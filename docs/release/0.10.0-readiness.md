## Included state

- `shiplog sources list` and `--json` show the seven canonical source keys
  without provider calls.
- `shiplog sources enable --source <name>` and `disable` preserve comments,
  provider records, and tokens while changing only `enabled`.
- The opt-in OpenAI-compatible LLM backend rejects malformed and non-HTTPS
  endpoints before building a client or sending an authorization header.
